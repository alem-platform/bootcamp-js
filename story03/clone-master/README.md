# The Clone Master

| Expected file   |
| --------------- |
| clone-master.js |

Your task is to write a function called `deepClone` that creates a deep copy of an object.

### Example:

```js
const original = { a: 1, b: { c: 2 } };
const clone = deepClone(original);

console.log(clone); // { a: 1, b: { c: 2 } }
```

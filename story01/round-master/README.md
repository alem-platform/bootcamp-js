# Round Master

### Instructions:

Create a function named `roundMaster` that takes a number and returns it rounded to two decimal places. Your goal is to ensure accuracy in calculations.

### Example:

```js
roundMaster(3.14159);
// Output: 3.14

roundMaster(123.456);
// Output: 123.46

roundMaster(9.1);
// Output: 9.10
```

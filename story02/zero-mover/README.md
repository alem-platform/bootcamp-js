# Zero Mover

| Expected file |
| ------------- |
| zero-mover.js |

### Instructions:

Write a function called `zeroMover` that moves all the zeroes in an array to the end, without altering the order of the non-zero elements.

### Expected funciton:
```js
function zeroMover(arr) {
  //
}
```

### Example:

```js
zeroMover([1, 0, 2, 0, 3]);
// Output: [1, 2, 3, 0, 0]

zeroMover([0, 0, 5, 6]);
// Output: [5, 6, 0, 0]
```

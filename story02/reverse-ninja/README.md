# Reverse Ninja

| Expected file    |
| ---------------- |
| reverse-ninja.js |

### Instructions:

Create a function called `reverseNinja` that takes a string and returns it reversed, using no built-in string methods.

### Example:

```js
reverseNinja("hello");
// Output: 'olleh'

reverseNinja("JavaScript");
// Output: 'tpircSavaJ'
```

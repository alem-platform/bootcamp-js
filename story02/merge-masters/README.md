# Merge Master

### Instructions:

Create a function called `mergeMaster` that takes two arrays as arguments and returns a new array that contains all the elements of both, without modifying the originals.

### Example:

```js
mergeMaster([1, 2], [3, 4]);
// Output: [1, 2, 3, 4]

mergeMaster(["a", "b"], ["c", "d"]);
// Output: ['a', 'b', 'c', 'd']
```
